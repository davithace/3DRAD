HighwayConstructionKit


HighwayConstructionKit is a road system for 3D Rad allowing you to easily assemble an highway supporting collision detection and specifically designed to work with other 3D Rad objects like cars.

Powerful editing features allow you to automatically snap road pieces together with absolute accuracy to guarantee perfectly looping tracks.

At the same time, the road elements don't have to be at fixed locations/orientations and their mutual position can be set freely where required.

After adding the HighwayConstructionKit object to your project, you can configure it by double-clicking it in the Object List.




Change

Click the Change button to specify the road element shape.

Fast rendering

If this option is checked, a faster rendering technique is used. This will improve frame rate on slower systems, but it may reduce rendering quality.

Note that on old systems that do not support hardware processed pixel shaders, this modality is automatically used.

Cast shadows

If this option is checked, the road element will cast a volumetric shadow. This may affect frame rate on slower systems.

Collision detection

If this option is checked, the road element will detect collision detection with all rigid-body based objects linked to it.
